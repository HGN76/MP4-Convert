# MP4-Convert
Convert Full HD MP4's to a specific resolution [in this case 1080x608] using HandBrakeCLI

## Requirements
- `HandBrakeCLI`

## How to install the CLI
```console
$ sudo apt-get install handbrake-cli
```

## How to run mp4-convert.sh
```console
~/$ chmod +x mp4-convert.sh
~/$ ./mp4-convert.sh
```
